### version:3.0 修改说明

    “占用”判断采用该视频帧前后多帧的IOU和box距离综合判断。

    “空位”状态在"驶出"状态出现后，逐秒判断之后满足条件的帧。
    发现输出的视频比原视频短，原因是未检测到的视频帧没输出，已修改。
    空位状态只会输出一次。
    输出的照片添加上了car_boundingbox。

    中间结果用pkl文件保存，加快第二次检测的速度。
